# Automação E2E (UI) - ServeRest E-commerce

Projeto de testes automatizados de interface (End-to-End) desenvolvido com **Cypress** e **JavaScript**, cobrindo os fluxos principais de navegação e autenticação do frontend da ServeRest.

## 🛠️ Tecnologias Utilizadas
* **Cypress** (v15+)
* **JavaScript**
* **API / Interface ServeRest** (https://front.serverest.dev)

## 🧪 Cenários Testados
1. **Validação de Elementos de Login:** Verifica se os campos de e-mail, senha e o botão de acesso estão visíveis na tela inicial.
2. **Tentativa de Autenticação Inválida:** Preenche os dados de credenciais não cadastradas e valida a exibição da mensagem de erro na interface.
3. **Navegação de Rota (Cadastro):** Valida a transição correta da tela de login para a tela de cadastro ao interagir com o link correspondente.
